How to use

  1. Download the HTML file.
  2. Double-click it to open in your browser, or drag it into Chrome or Edge.
  3. Select the recording.
  4. Click Create Transcript.
  5. Download TXT, SRT, VTT, or JSON when it finishes.
Recommended browser: recent Chrome or Edge. WebGPU can be faster, but if it fails, switch Backend to WASM.

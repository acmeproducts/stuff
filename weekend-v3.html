About Weekend v3 Reporting Upgrade

Goal

`weekend-v3.html` starts from the `weekend-v2.html` consolidation as the baseline, then adds a stronger reporting layer inspired by `compound.html` without re-merging older weekend variants.

Superset Coverage

Weekend v3 preserves the v2 workflow, analytics, maintenance CRUD, tag workflow, planning sandbox, small multiples, and import/export model while upgrading the standard drill-down report with summary cards, action buttons, sticky report totals, and print/PDF support adapted from `compound.html`.

Reporting + Settings Notes

  • The drill-down report now uses `compound.html` patterns for a stronger summary header, a dedicated report action bar, a stickier tabular report treatment, and a print-first PDF workflow.
  • Existing Weekend v2 reporting logic remains the baseline: projections, capital-adjustment handling, clarifying notes, and chart drill-downs are unchanged unless a stronger reporting presentation pattern replaced them.
  • Visual presentation settings now persist beyond accent color. Theme, density, font scale, contrast, chart legend visibility, report notes visibility, badge visibility, and striped report tables are all saved to localStorage and travel in Settings export/import JSON.
  • Settings are intentionally centralized so there is one cohesive flow: the visual settings modal controls appearance, the system settings modal handles data/settings export-import, and report actions live inside the drill-down report instead of spawning duplicate reporting screens.
  • The About tab doubles as in-app documentation so operators can see which reporting affordances came from `compound.html` and which readability settings are persisted.

Unified Data Notes

  • books[]
  • portfolios[]
  • chunks[] => { type, capital, duration, targetPct, rungs[], tags[], entryPct, exitStrategy }
  • actuals[] => { value, adjustment, note, date }

Import logic auto-normalizes older weekend data so previously saved structures continue to work inside the consolidated app. Weekend v3 settings export/import also includes the full reporting/appearance profile described above.
